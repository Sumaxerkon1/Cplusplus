# C++ Playground: From Zero to OOP

51 standalone C++ scripts I wrote while learning the language, going from the absolute basics all the way to OOP. Each file covers one concept and is meant to be a quick reference - if you want to see how something works without reading through a tutorial, just open the relevant file.

Everything is in chronological order so you can also just go through them one by one if you want to follow along.

## How to run

All files are standard C++17, no external libraries needed.

**Option 1 - VS Code (easier)**

Open the folder in VS Code, install the C/C++ extension by Microsoft, then open any `.cpp` file and press F5 to compile and run it.

**Option 2 - Terminal**

```
g++ -std=c++17 "filename.cpp" -o output.exe
./output.exe
```
